# Dantist1999 Hero

Самодостаточный hero-блок для ручного переноса в основной проект.

## Файлы

- `index.html` - разметка блока и демо-страницы.
- `styles.css` - вся визуальная часть и адаптивы.
- `script.js` - параллакс изображения и валидация формы.
- `assets/layer-1.webp` ... `assets/layer-4.webp` - слои изображения из Figma.

## Интеграция

Для переноса достаточно взять секцию `.dantist-hero`, подключить стили из `styles.css`, скрипт из `script.js` и сохранить относительные пути к изображениям в `assets`.
